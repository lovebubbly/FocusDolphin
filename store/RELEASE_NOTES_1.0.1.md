# Focus Dolphin 1.0.1 Release Notes

Prepared by **OpenAI Codex (GPT-5)** for product owner **Choi Yunseong (최윤성)** on **2026-07-15 KST**.

## English

Fixed two issues in Gentle Reminder mode:

- The prompt now renders at the correct size on sites with custom root font sizing, including YouTube.
- After completing the prompt, it stays dismissed while navigating between pages on the same hostname during the current focus session.

No permissions were added, and Focus Dolphin remains local-first.
